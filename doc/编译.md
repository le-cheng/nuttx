
# 编译方法一: make编译
## 初始化
```bash
cd nuttx
make clean && ./tools/configure.sh -l sim:dualscreen
```
## 编译
```bash
make -j$(nproc)
```
## 清理
```bash
make clean
make distclean
```

## 运行
```bash
./nuttx
```

# 编译方法二: cmake编译
## 编译

```bash
cd nuttx
rm -rf build/ && cmake -B build -DBOARD_CONFIG=sim:dualscreen -GNinja
```

```bash
ninja -C build -j$(nproc) && ./build/nuttx
```
## 配置
```bash
ninja -C build menuconfig
```

## 清理
```bash
ninja -C build clean
```

## 运行
```bash
./build/nuttx
```
